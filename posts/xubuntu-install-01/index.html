xubuntu安装环境（一）

2018/08/09

简单记录xubuntu的安装注意事项及中文环境配置

ubuntu作为linux的一个重要版本，它本身又分出了众多版本:

  • xubuntu
  • ubuntu-mate
  • Lubuntu
  • Kubuntu

我选用了xubuntu，因为它用到XFCE窗口管理器，小巧快速，在低配置的电脑上也可流畅运行。

xubuntu 18.04.1安装

  1. 将制作好的xubuntu启动u盘插入，开机，进入bios画面，设置boot（启动优先级），设置成第一个 uefi （启动的类型）
  2. 能从u盘启动后，选择第二项，install xubuntu
  3. 然后正式进入安装 xubuntu linux 的程序
    • 语言选择
    • 网络不要设定，断网环境下安装，这样直接安装u盘上的软件包，速度快
    • 时区
    • 分区设定，初学先简化，清除整个硬盘并安装xubuntu，（若是有windows的话，不想删除，则不能选此项安装，需选择手动分区）
    • 用户名设定（具有sudo权限）
  4. 开始安装，一直等待到安装完成

xubuntu环境设定

  1. 找到 “软件和更新” 菜单，选择下载服务器，中国服务器，选择“清华”的镜像站点
  2. 语言支持，中文语言包不完整，提示安装
  3. 安装完成中文语言包之后，注销后重新登录，设定输入法fcitx，注销后登录即可生效
  4. 安装新立得系统软件包 sudo apt install synaptic (apt与apt-get一样）
  5. 用新立得来安装git（分布式源码版本管理）, chromnium（谷歌浏览器的开源版本）
