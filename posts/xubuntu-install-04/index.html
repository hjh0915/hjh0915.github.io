xubuntu安装环境（四）

2018/08/10

python开发用的编辑器，可以用vim, gedit, sublime text等等，我选visual studio code。

visual studio code安装

  1. 从微软的网站下载，网址链接，下载ubuntu的deb包，然后双击安装

  2. 运行进入

    • 安装中文语言包，扩展搜索chinese，即可找到
    • 配置： zoomLevel那行是同步放大，一般设定为1即可，其他是为了打开文件夹后，排除一些与开发无关的文件显示
{
"window.zoomLevel": 2,
"files.exclude": {
    "**/.git": true,
    "**/.svn": true,
    "**/.hg": true,
    "**/CVS": true,
    "**/.DS_Store": true,
    "*.pyc": true,
    "**/.vs": true
    },
}
