xubuntu安装环境（三）

2018/08/09

python 2.x 和 python 3.6.6 混合使用的方式

pyenv 安装

多版本python的管理器，pyenv 用pyenv来安装python 3.6.6环境

  1. 先安装必备的一些xubuntu系统包 具体参见：网址链接

    sudo apt-get install -y make build-essential libssl-dev zlib1g-dev libbz2-dev \
libreadline-dev libsqlite3-dev wget curl llvm libncurses5-dev libncursesw5-dev \
xz-utils tk-dev libffi-dev

  2. 采用pyenv-installer安装 具体参见：网址链接

    在shell下，先运行

    export http_proxy=http://localhost:8118 
export https_proxy=http://localhost:8118

    就可以利用之前的privoxy+shadowsocks建立起来的代理通道

    再运行

    curl -L https://github.com/pyenv/pyenv-installer/raw/master/bin/pyenv-installer | bash

  3. 配置 .bashrc 增加如下配置:

    export PATH="~/.pyenv/bin:$PATH"
eval "$(pyenv init -)" \\
eval "$(pyenv virtualenv-init -)"

    然后 source .bashrc 生效刚才的配置

  4. 用pyenv安装python 3.6.6

    • 先确保pyenv能否正常运行
    • 然后同样利用privoxy+shadowsocks的代理通道，运行 pyenv install -v 3.6.6
    • 等待python 3.6.6下载、编译、安装等一系列过程结束

  5. 正式建立自己的python 3.6.6环境

    mkdir myenv
cd myenv
pyenv local 3.6.6

    在myenv目录及其子目录下，python版本切换到3.6.6，不在myenv目录的其他目录则仍然是python 2.7
